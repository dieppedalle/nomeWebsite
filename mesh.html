NOME Language Specification
Mesh Generators

Tunnel

NOME Definition

A tunnel statement generates a cylindrical surface that can be modeled by the inner part of a torus.

tunnel id (n ro ratio h) endtunnel

The n, ro, ratio, and h parameters of a tunnel are numbers that need to be separated by space.

Example

tunnel t1 (6 2 3 4) endtunnel

Defines a tunnel named t1 with parameters n=6, ro=2, ratio=3, and h=7. n represents the number of vertices along the rim of the tunnel. ro represents the radius of the tunnel. ratio represents the size of the middle rim compared to the outside rims. h is the height of the tunnel from the middle rim to the outside rim.

Funnel

NOME Definition

A funnel statement generates a border curve that is constrained to be circular. They can be obtained by truncating a cone.

funnel id (n ro ratio h) endfunnel

The n, ro, ratio, and h parameters of a funnel are numbers that need to be separated by space.

Example

funnel f1 (6 2 3 4) endfunnel

Defines a funnel named f1 with parameters n=6, ro=2, ratio=3, and h=7. n represents the number of vertices along the rim of the tunnel. ro represents the radius of the tunnel. ratio represents the size of the middle rim compared to the outside rims. h is the height of the tunnel from the middle rim to the outside rim.

Circle

NOME Definition

A circle statement generates a set of edges and vertices forming a circle given a radius and a number of slices.

circle id (num radius) endcircle

The num, and radius parameters of a circle are numbers that need to be separated by space.

Example

circle c1 (5 10) endcircle

Defines a circle named c1 with parameters num=5, and radius=10.
