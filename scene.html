NOME Language Specification
Scene Parameters

Single-Line Comment

NOME Definition

The user can enter the # character to add a line comment. When the parser encounters the # symbol, the characters on the same line after the symbol will be ignored.

# Any ASCII characters on the same line after pound sign ignored

Example

# This is a comment

Multi-Line Comment

NOME Definition

The user can enter the (* *) characters to add a multi-line comment. When the parser encounters the (* symbol it ignores every character until *). Nested comments are allowed.

(* Any ASCII characters on the different lines are ignores *)

Example

(* This is a comment *)

delete

NOME Definition

Deletes a face from a geometric construct.

delete face face_id1 endface ... face face_idN endface enddelete

Each of the face_id must have already been created in order to delete them.

Example

delete face cube1_c4 endface face cube1_c2 endface face cube1_c3 endface enddelete

This method will delete the faces cube1_c4, cube1_c2, and cube1_c3.

Subdivision

NOME Definition

A subidision statement is used to subdivide a mesh.

subdivision id type SLF_CATMULL_CLARK subdivisions subdivVal endsubdivision

The subdivVal represents the maximum number of subdivisions for the slider

Example

subdivision subdiv type SLF_CATMULL_CLARK subdivisions 5 endsubdivision

Defines a subdivision slider that has a maximum of 5 subdivisions.

Offset

NOME Definition

An offset statement is used to offset a mesh once it has been merged.

offset id min minVal max maxVal step stepVal endoffset

The minVal represents the minimum value of the slider. The maxVal represents the maximum and the step represents the different in value between two slider values.

Example

offset of min 0.0 max 0.4 step 0.02 endoffset

Defines an offset slider with minimum 0, step 0.02, and maximum of 0.4.
