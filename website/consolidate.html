Consolidating vs. Merging

Each time a user adds a polyline or a face to mesh via the user interface, he/she needs to press the consolidate button in order to integrate it permanently into the mesh.

User needs to press on the merge button before subdividing or offsetting a mesh. The merge button links all the faces, vertices, and edges together from the mesh.
