NOME Language Specification
Spline Nodes

BSpline

NOME Definition

A bspline statement is used to define a curve that approximates a list of control points.

bspline{order} id ( point_idlist ) closed endbspline{order}

The point_idlist defines a list of control points. The closed parameter defines whether or not a curve is closed.

Example

bspline3 bsp (p0 p1 p2 p3 p4 p5 p6) endbspline3

Defines a bspline bsp with order 3 and with control points p0, p1, p2, p3, p4, p5, and p6.

Bezier Curve

NOME Definition

A bezier curve statement is used to define a curve that interpolates the end points and approximates the rest of the points from a list of control points.

beziercurve id ( point_idlist ) slices numSlices endbeziercurve

The point_idlist defines a list of control points. The numSlices parameter defines the number of slices in the bezier curve.

Example

beziercurve bz (p0 p1 p2 p3 p4 p5 p6) slices 10 endbeziercurve

Defines a bezier curve bz with 10 slices and with control points p0, p1, p2, p3, p4, p5, and p6.
